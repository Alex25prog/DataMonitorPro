# DataMonitor Pro

**DataMonitor Pro** — это мощное приложение для мониторинга и анализа данных в реальном времени. Оно принимает данные по сети, отображает их на графике и в таблице, сохраняет историю в базу данных и позволяет экспортировать результаты.

---

## 🚀 Возможности

- 📡 **Прием данных** — TCP/WebSocket сервер для приема данных в формате JSON
- 📈 **Визуализация** — график реального времени и таблица с данными
- 💾 **Сохранение истории** — все данные сохраняются в SQLite
- 🔍 **Фильтрация** — по типу данных и временному диапазону
- 📊 **Статистика** — расчет минимума, максимума, среднего
- 📤 **Экспорт** — сохранение данных в CSV и PDF

## 📦 Установка и сборка

### Требования

- Qt 6.10 или выше (с компонентом MinGW 64-bit)
- CMake 3.20 или выше
- Git (опционально)

### Сборка проекта

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/DataMonitorPro.git
cd DataMonitorPro

# Создание папки для сборки
mkdir build
cd build

# Конфигурация CMake
cmake ..

# Компиляция
cmake --build . --config Release
